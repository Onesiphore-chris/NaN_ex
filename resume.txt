                // RESUME DE GIT 

C'est quoi Git ? Git est un système de contrôle de version open source.
Concrètement, c'est un outil qui te permet de traquer tous les fichiers 
de ton projet. Chaque modification de fichier 
est alors détectée par Git et versionnée dans une version instantanée

 a ce effet plusieur commande son definie pour , 

 -- l'insertion
 -- l'affichage
 -- la recuperration 
 
//  Démarrer un dépôt Git

Cloner un dépôt existant
$ git clone ssh://chris@serveur/git/Nan_exo.git
$ git clone

II. Enregistrer des modifications dans un dépôt
a. Vérifier l’état des fichiers
$ git status

II. Enregistrer des modifications dans un dépôt
c. Valider les modifications
$ git commit –m « Mon premier commit» 
La commande «commit» 

// Visualiser l’historique des validations
$ git log

// Enregistrer des modifications dans un dépôt

Pousser son travail sur un dépôt distant
$ git push origin master

La commande «push» sert à envoyer tout les «commits» 

Récupérer et tirer depuis des dépôts distants
$ git pull 
La commande «pull» permet de mettre à jour votre dépôt 

Créer une nouvelle branche nommée « feature_x »
$ git branch feature_x
 Basculer vers une branche existante
$ git checkout feature_x


//  Retourner sur la branche principale
$ git checkout master

 // Supprimer la branche
$ git branch –d feature_x

//  Incorporation des modifications d’une branche 
$ git merge <branche>
